Cheat Sheet - HTML

<!DOCTYPE html>

Indica al navegador que el documento está escrito en HTML

<!DOCTYPE html>
<html>

Etiqueta raíz del documento HTML. Todo el contenido va dentro de ella.

<html lang="es">
...</html>
<head>

Contiene metadatos del documento, como el título, enlaces a estilos o scripts.

<head>
         <title>Mi página</title>
</head>
<body>

Contiene todo el contenido visible de la página web, como texto, imagenes, videos, etc.


        <body>
        <h1>Hola Mundo</h1>
        </body>
<title>

Define el título de la página que aparece en la pestaña del navegador.

<title>Mi Página Web</title>
<p>

Define un párrafo de texto, se usa para separar bloques de texto.

<p>Este es un párrafo.</p>
<header>

Seccion de cabecera de la pagina, contiene normalmente un logo, titulo o menu de navegacion.

<header>Logo de la pagina</header>
<nav>

Define un menú de navegación con enlaces a otras partes del sitio.

<nav><a href="#">Inicio</a></nav>
<main>

Contenido principal de la pagina, solo debe existir uno por pagina


            <main>
            <h1>Bienvenido</h1>
            <p>Contenido principal</p>
            </main>
<aside>

Contenido lateral o complementario, como barras laterales o publicidad.

<aside>Enlaces relacionados </aside>
<footer>

Pie de página con información de contacto, copyright o enlaces legales.

<footer>© 2025 Mi sitio web </footer>
<h1> - <h6>

Encabezados de importancia decreciente. h1 es el más importante y h6 el menos.


        <h1>Título Principal</h1>
        <h2>Subtítulo</h2>
<span>

Contenedor en línea para aplicar estilos o agrupar texto.


            <main>
            <h1>Bienvenido</h1>
            <p>Contenido principal</p>
            </main>
<div>

Contenedor en bloque para agrupar secciones de contenido.


        <div><p>Texto dentro de un div</p> </div>
<a>

Crea un enlace.

<a href="https://www.google.com">Ir a Google</a>
<ol>, <ul>, <li>

Etiquetas para listas.
<ol> crea una lista ordenada (numerada).
<ul> crea una lista desordenada (con viñetas).
<li> define cada elemento de la lista. 

<ol>
  <li>Primero</li>
  <li>Segundo</li>
</ol>

<ul>
  <li>Uno</li>
  <li>Dos</li>
</ul>
<dl>, <dt>, <dd>

Etiquetas para listas de definiciones.
<dl> lista de definiciones.
<dt> término.
<dd> definición o descripción. 

<dl>
<dt>HTML</dt>
<dd>Lenguaje de marcado para la web</dd>
</dl>
<table>, <tr>, <td>, <th>

Etiquetas para tablas.
<table> crea la tabla.
<tr> fila.
<td> celda normal.
<th> celda de encabezado. 

<table>
  <tr>
    <th>Nombre</th>
    <th>Edad</th>
  </tr>
  <tr>
    <td>Ana</td>
    <td>20</td>
  </tr>
</table>
<thead>, <tbody>, <tfoot>

Secciones de una tabla.
<thead> encabezado.
<tbody> cuerpo.
<tfoot> pie de tabla. 

<table>
  <thead>
    <tr><th>Columna</th></tr>
  </thead>
  <tbody>
    <tr><td>Dato</td></tr>
  </tbody>
  <tfoot>
    <tr><td>Total</td></tr>
  </tfoot>
</table>
<img>, <audio>, <video>, <source>

Etiquetas para medios.
<img> imagen.
<audio> sonido.
<video> video.
<source> fuente del archivo multimedia. 

<img src="foto.jpg" alt="Descripción">

<audio controls>
 <source src="audio.mp3" type="audio/mpeg">
</audio>

<video controls>
<source src="video.mp4" type="video/mp4">
</video>
<form>

Define un formulario para recolectar datos.

<form action="/enviar" method="post">
        <input type="text" name="nombre">
        <input type="submit" value="Enviar">
</form>
<input>

Campo de entrada en formularios (texto, número, correo, etc.).

<input type="text" placeholder="Escribe tu nombre">
<input type="email" placeholder="Correo">
<select>, <option>

Menú desplegable.
<select> define la lista.
<option> cada opción del menú. 

<select>
    <option>Opción 1</option>
    <option>Opción 2</option>
</select>
<textarea>

Área de texto multilínea en formularios.

<textarea rows="4" cols="30">Escribe aquí...</textarea>
<label>

Etiqueta asociada a un campo de formulario, mejora la accesibilidad.

<label for="nombre">Nombre:</label>
<input id="nombre" type="text">
